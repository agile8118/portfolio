What I do

Front-end Development: I develop responsive, modern looking front ends. Most of the time I develope my own CSS frameworks, and in cases that the project needs to be done as fast as possible, I use CSS frameworks written by others like bootstrap, material UI, etc.  And as for JavaScript, I am an expert in React and I use it all the time to power rich front end applications.

Back-end Development: For backend servers most of the time I use Node.js, Python, and C++ for cases that performance is critical. Security, scalability, and performance are things I keep in mind when I am developing backend systems.

Full-stack Development: I can build complete websites and applications, ready to use with all the concerns in front-end, back-end, and general websites stuff. I can deploy and maintain complex web applications and keep track of their security, SEO, performance, cross-browser support and more.

DevOps: I have a decent experience working with cloud infrastructures and technologies, deploy automation, system management, and continuous integration. I can deploy, maintain and scale huge web applications and servers on the cloud. I mostly use AWS to manage Linux servers, databases, storage, security and networking.

General Terminal App Development: I build terminal-based applications for Unix-based and other platforms for automation, data analysts, web scraping, modifying images and many other tasks. I use C++ and Python for developing these kinds of useful software.
